# 介绍

这是一个非常小巧、美观的PC端小说阅读器。

同时为了节约空间，软件所使用的小说文件格式为：txt文本经bzip2算法压缩得到的文件。

暂时只支持window，未来看情况支持linux、mac、android等等

其中novels文件夹中为基本测试小说，均为网络收集，如有侵权，可联系我进行删除
